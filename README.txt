lista de Tarefas (Java no console)

Esse projeto é uma lista de tarefas bem simples feita em Java, para rodar direto no terminal.  
A ideia é ter um lugar rápido para anotar o que precisa fazer, marcar quando estiver fazendo e quando terminar.


 O que dá pra fazer

- Adicionar tarefa com nome, descrição e status.
- Listar todas as tarefas que já foram cadastradas.
- Remover uma tarefa pelo ID.
- Mudar o status de uma tarefa (por exemplo, de DISPONIVEL para FAZENDO ou FEITA).

Status disponíveis:

- `DISPONIVEL` – ainda não comecei.
- `FAZENDO` – estou trabalhando nela.
- `FEITA` – missão cumprida.

 Como fazer para rodar

1. Tenha o Java (JDK 8 ou superior) instalado.
2. Abra o terminal na pasta do projeto e rode:

   ```bash
   javac *.java (java.main)
